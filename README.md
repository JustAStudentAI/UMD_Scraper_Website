# UMD Dining Hall Menu Web Scraper

## https://web-scraper-webapp.onrender.com/scrape

This website is designed to display the University of Maryland (UMD) dining hall menus. Users can select a dining hall and a meal time to view the available menu options.  The menus are scraped from the official UMD dining services website using a java application.


## Features

- Dynamically fetches the menu for the current day from the UMD dining services website.
- Allows users to select a dining hall and meal time.
- Displays the scraped menu information on a web page.


## Technologies

- Spring Boot
- Jsoup for HTML parsing
- Thymeleaf for views
- Render to deploy website
